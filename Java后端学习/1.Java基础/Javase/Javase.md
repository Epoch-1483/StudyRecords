[[Java后端学习笔记]]
Java SE（Java Platform, Standard Edition，Java 标准版）是 Java 平台的基础版本，主要用于开发桌面应用、命令行工具以及一些简单的服务器端应用程序。它是 Java 技术的核心，为其他 Java 平台（如 Java EE / Jakarta EE 和 Java ME）提供了基础支持。

### 主要特点：

1. **跨平台性**  
    Java SE 基于“一次编写，到处运行”（Write Once, Run Anywhere）的理念，通过 Java 虚拟机（JVM）实现跨操作系统运行。
    
2. **核心类库**  
    提供了丰富的标准 API，包括：
    
    - 基础语言类（如 `String`、`Math`、集合框架等）
    - 输入/输出（I/O）
    - 网络编程
    - 多线程
    - 反射、泛型、注解
    - 图形用户界面（GUI，如 Swing 和 AWT）
    - 安全、国际化、日期时间处理等
3. **开发工具**  
    包含 JDK（Java Development Kit），提供编译器（javac）、运行环境（JRE）、调试工具（jdb）、打包工具（jar）等。
    
4. **面向对象**  
    Java SE 完全面向对象，支持封装、继承、多态等特性。
    
5. **自动内存管理**  
    通过垃圾回收机制（Garbage Collection）自动管理内存，减少内存泄漏风险。
    

### 应用场景：

- 桌面应用程序（如使用 JavaFX 或 Swing 开发的 GUI 程序）
- 工具类软件（如 IDE 插件、命令行工具）
- 学习 Java 编程的入门平台
- 作为 Web 应用或企业级应用（Java EE/Jakarta EE）的基础

### 版本演进：

Java SE 自 1996 年发布 1.0 版本以来持续更新，目前主流版本包括 Java 8（长期支持）、Java 11（LTS）、Java 17（LTS）和最新的 Java 21（LTS）。每个新版本通常会引入新特性、性能优化和安全增强。

---

简而言之，**Java SE 是学习和使用 Java 的起点，也是构建更复杂 Java 应用的基础**。
