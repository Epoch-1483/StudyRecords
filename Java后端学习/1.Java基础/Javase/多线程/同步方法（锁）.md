synchronized 关键字加到方法上
![[Pasted image 20251101150806.png]]
特点一：
	同步方法是锁住方法里面的所有的代码
特点二：
	锁对象不能自己指定
非静态 ： this 
静态 ：  当前类的字节码文件对象

ctrl + alt  + M  选中代码块抽取成一个方法
## lock锁：
```
static Lock lock = new ReentrantLock();

lock.lock();

lock.unlock();
```
