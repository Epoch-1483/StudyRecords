细节1：
	1.如果我们没有给线程设置名字，线程也是有默认的名字的
	2.如果我们要给线程设置名字，可以用set方法也可以用构造方法设置
细节2：
	当jvm虚拟机启动之后，会自动的启动多条线程
	 其中一条线程就叫做Main线程
	 它的作用就去调用Main方法，并执行里面的代码
	 在以前，我们写的所有代码，其实都是运行在Main线程中的
细节3：
	1.哪条线程执行到这个方法，那么哪条线程就会在在这里停留对应的时间
	2.方法的参数：就表示睡眠时间。单位毫秒
	3.当时间到了以后，线程会自动的醒来，继续执行下面的代码
