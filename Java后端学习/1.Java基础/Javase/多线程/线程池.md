	1.用到线程的时候创建
	2.用完之后线程消失

如果提交任务时，池子中没有空闲线程，也无法创建新的线程，任务机会排队


## 自定义线程池：
	1.核心线程
	2.临时线程
什么时候用临时线程排队排满了还有多的
![[Pasted image 20251102112339.png]]


![[Pasted image 20251102112234.png]]

```
ThreadPoolExecutor pool = new ThreadPoolExecutor(  
        // 核心线程数量  
        3,  
        // 最大线程数，不能小于0，最大数量 >= 核心线程数量  
        6,  
        // 空闲线程的最大存活时间 值  
        60,  
        // 时间的单位  
        TimeUnit.SECONDS,  
        // 任务的队列  阻塞队列
        new ArrayBlockingQueue<>(3),  
        // 创建线程工厂  
        Executors.defaultThreadFactory(),  
        // 任务的拒绝策略  
        new ThreadPoolExecutor.AbortPolicy()  
);
```
