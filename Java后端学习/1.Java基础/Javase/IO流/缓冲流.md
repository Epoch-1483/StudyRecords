### 🌰 举个例子：快递站取包裹

想象一下，你每天都要去快递站取包裹。

- **普通方式（没有缓冲）**：你每收到一个包裹，快递员就立刻跑一趟，把包裹送到你手上。如果一天有100个包裹，他就得跑100趟，非常累，效率也低。
    
- **缓冲方式（有缓冲）**：快递员先把100个包裹都堆在一个“暂存区”（这就是缓冲区），等攒够了一堆，或者到了某个时间点，再一次性送给你。这样他只跑一趟，效率高多了！
