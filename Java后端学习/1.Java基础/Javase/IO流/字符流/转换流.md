## 是字符流和字节流之间的桥梁
![[Pasted image 20251028173847.png]]


![[Pasted image 20251028174818.png]]

- `InputStreamReader`：
				- 把**字节流**转换成**字符流**（读的时候用）
- `OutputStreamWriter`：
				- 把**字符流**转换成**字节流**（写的时候用）

**转换流就是用来“翻译”的**：  
	把字节流“翻译”成字符流（读文本时），或者把字符流“翻译”回字节流（写文本时），并且**可以指定编码格式**！
## 生活中的例子：
想象你收到一封外文信（比如德语），但你只会中文：

- **字节流** 就像你拿到的是一堆看不懂的德语文字（原始字节）。
- **转换流** 就像一个翻译官，他懂日文也懂中文，还能告诉你：“这封信用的是 UTF-8 编码”。
- **字符流** 就是你最终看到的中文内容。

没有这个“翻译官”（转换流），你就可能把德语错看成乱码！

```
InputStreamReader isr = new InputStreamReader(new FileInputStream("myio\\gbkfile.txt),"GBK");
```


2.字节流想要使用字符流中的方法
```
FileReader fr = new FileReader("myio\\gbkfile.txt",Charset.forName("GBK"));
int ch;
while((ch = fr.read()) != -1 ){
	sout((char)ch)
}
fr.close;
```
## 转换流练习使用JDK11 之后的方案：
```
FileReader fr = new FileReader("myio\\b.txt", Charset.forName("GBK")); 
FileWriter fw = new FileWriter("myio\\e.txt", Charset.forName("UTF-8"));
int b; 
while ((b = fr.read()) != -1) {
	fw.write(b); 
} 
fw.close();
```
