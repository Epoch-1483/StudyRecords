弊端：
拷贝慢因为一次读取一个字节
![[Pasted image 20251026194541.png]]


解决办法：一次读取多个字节

![[Pasted image 20251026194801.png]]

```
int len;
byte[] bytes = new byte[2];
len = fis.read(bytes);
System.out.println(new String(bytes))
```


每次获取的数据都会覆盖上次的，没有被覆盖的话数据保持原样


![[Pasted image 20251026195853.png]]

解决上次的残留数据方法：
在输出语句中的数组对象bytes 后面加上0和len，0表示从第一个开始，表示获取几个
```
int len;
byte[] bytes = new byte[2];
len = fis.read(bytes);
System.out.println(new String(bytes，0，len))
```
