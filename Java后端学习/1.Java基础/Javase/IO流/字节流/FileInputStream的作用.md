`InputStream` 是所有**字节输入流**的抽象基类，用于从**源**（如文件、网络、内存等）读取字节数据。
操作本地文件的字节输入流，可以把==本地文件中的数据==读取到==程序==中来

1.书写步骤：
1. 创建字节输入流对象
2. 读数据
3. 释放资源

3.循环读取：
用while循环因为不知道文件中的数据有多少

示例：
```
		int b;  
		while ((b = fos.read()) != -1){  
		    System.out.println((char)b);  
		}
```

read ：表示读取数据，而且是读取一个数据就移动一次指针


- **字节输入流**：用于**读取**原始字节，适用于文件读取、网络接收、对象反序列化等。
- **字节输出流**：用于**写入**原始字节，适用于文件保存、网络发送、对象序列化等。
