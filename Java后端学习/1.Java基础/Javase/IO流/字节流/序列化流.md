### 🌰 举个例子：打包行李去旅行

想象一下你要搬家或者去旅行，你有一堆东西：衣服、书、玩具、锅碗瓢盆……  
但你不能把这些东西散着搬走，对吧？你需要把它们**打包进箱子**，这样就能方便地运输。

到了新家之后，你再把箱子打开，把里面的东西**一件件拿出来还原**，恢复成原来的样子。

---

### 💡 在 Java 中：

- **对象**就像是你房间里那些“活生生”的东西（比如一个用户对象，里面有名字、年龄、地址等）。
- **序列化（Serialization）** 就是把这些“活的对象”**打包成一串字节**（就像打包进行李箱），这样就可以：
    - 存到硬盘上（比如保存用户设置）
    - 通过网络发送给别人（比如客户端和服务器之间传数据）
- **反序列化（Deserialization）** 就是在另一端或以后某个时间，**把这串字节重新变回原来的对象**（就像拆开箱子，把东西还原）。


- `ObjectOutputStream`：用来**写对象**（序列化），把对象变成字节流，可以写到文件或网络。
- `ObjectInputStream`：用来**读对象**（反序列化），从字节流中恢复出原来的对象。


### ⚠️ 注意一点：

不是所有对象都能被序列化！  
Java 要求：**类必须实现 `Serializable` 接口**，就像贴个标签说：“我同意被打包！”

```
public class User implements Serializable { 
	private String name; 
	private int age;
	 // ... 
}
```

## Java 的序列化流，就是用来把对象“变成字节”保存或传输，之后再“变回对象”的工具。
